# Itinerario Puglia 🇮🇹

Página web interactiva con el itinerario de 8 días por Puglia (4–11 de julio):
mapa con todos los lugares, ruta día por día, playas, excursiones, restaurantes
y una sección de gastronomía por zona (Michelin, cafés y especialidades típicas).

Es una página estática autocontenida (`index.html`). El mapa usa Leaflet +
OpenStreetMap, así que necesita conexión a internet para cargar los tiles.

## Deploy
Hosteada en Vercel (plan Hobby, gratis). Cada push a la rama principal
re-despliega automáticamente.
